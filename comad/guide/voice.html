Claude Code의 루트 CLAUDE.md에 정의된 트리거들이 특정 키워드를 감지해 자동 워크플로우를 발동합니다. "상황 검토해봐" 한마디로 코드베이스 진단이 실행되고, "저장해줘"로 세션 인수인계 문서가 만들어집니다.

CLAUDE.md 6 triggers natural language non-developer friendly

개요

Claude Code는 작업 시작 시 루트 CLAUDE.md를 읽고 그 내용을 시스템 프롬프트로 내재화합니다. Voice는 이 메커니즘을 활용해 자주 쓰는 워크플로우를 한국어 트리거 키워드로 자동화합니다.

동작 모델:

Claude Code 시작
  ↓
CLAUDE.md 로드 (Nexus + OMC + comad Voice)
  ↓
사용자 입력
  ↓
키워드 매칭 검사
  ↓
  매칭 → 해당 트리거 skill/agent 자동 실행
  미매칭 → 일반 대화

Brain/Ear/Eye 연동

Voice 트리거는 다른 comad 모듈을 호출할 수 있습니다:

  • T15 검토 시 comad_brain_impact로 레퍼런스 엔티티 영향도 분석
  • T5 세션 저장 시 주요 결정을 Brain Neo4j의 Claim 노드로 자동 저장
  • T0 온보딩 시 comad_brain_search로 유사 프로젝트 기존 패턴 조회
