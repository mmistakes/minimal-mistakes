# Handoff Index

## handoff 위치

- handoff 문서는 모두 `project-docs/handoff/` 아래에 둔다.

## 현재 활성 handoff

- `HANDOFF_RUST_CURRICULUM.md`: Rust 시리즈 전체 순서, 공백, 메타데이터 규칙
- `HANDOFF_NEXT_RUST_POST.md`: 바로 다음 Rust 포스트 1편 제작용 실행 메모

## 권장 읽기 순서

- Rust 시리즈를 이어받을 때는 먼저 `HANDOFF_RUST_CURRICULUM.md`를 본다.
- 실제 작성 직전에는 `HANDOFF_NEXT_RUST_POST.md`를 본다.

## 갱신 조건

- 새 Rust 글을 발행한 경우
- 시리즈 번호, 주제, 순서를 바꾼 경우
- 제목, permalink, SEO 설명, 검증 기준이 바뀐 경우
