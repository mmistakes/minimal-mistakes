Quick Start

Up and running in 4 commands

# 1. Clone the repo
git clone https://github.com/kinkos1234/comad-world.git

# 2. Enter the directory
cd comad-world

# 3. Pick a preset (or create your own)
cp presets/ai-ml.yaml comad.config.yaml

# 4. Install & start
./install.sh

Requires: Node.js 20+, Python 3.13+, Neo4j 5, Bun. The install script handles dependency setup, Neo4j schema initialization, and cron configuration. See the README for detailed instructions.
