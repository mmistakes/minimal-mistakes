# 월 입력받아 계절 출력하기

**문제 설명**  
평가를 문자(A, B, C, D, ...)로 입력받아 내용을 다르게 출력해보자.

평가 내용  
평가 : 내용  
A : best!!!  
B : good!!  
C : run!  
D : slowly~  
나머지 문자들 : what?  
<br>

**입력**  
영문자 1개가 입력된다.  
(A, B, C, D 등 문자 1개가 입력된다.)  
<br>

**출력**  
문자에 따라 다른 내용이 출력된다.  
<br>

**입력 예시**  
A  
<br>

**출력 예시**  
best!!!

---

<br>

> **Source**

```python
n = input()

if n=='A':
    print('best!!!')
elif n=='B':
    print('good!!')
elif n=='C':
    print('run!')
elif n=='D':
    print('slowly~')
else:
    print('what?')
```
