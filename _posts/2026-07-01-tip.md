안녕하세요! 리눅스 환경이나 라즈베리파이 터미널에서 작업을 하다 보면, 평소보다 시스템이 눈에 띄게 느려지거나 마우스 커서가 버벅거리는 현상을 겪을 때가 있습니다. 
윈도우라면 '작업 관리자(`Ctrl + Shift + Esc`)'를 켜서 범인을 찾겠지만, 마우스가 멈춘 상황이거나 CLI 터미널 환경이라면 어떻게 해야 할까요? 
오늘은 터미널 명령어 몇 줄로 시스템 병목의 원인을 정확히 진단하고, 특히 리소스를 많이 먹는 '크로미움(Chromium) 브라우저'를 쌩쌩하게 만드는 최적화 해결 방법을 알아보겠습니다.
---
## 1단계: 터미널에서 시스템 상태 점검하기
시스템이 느려졌을 때 당황하지 않고 가장 먼저 확인해야 할 핵심 명령어 3가지입니다.
### ① 종합 실시간 모니터링: `htop`
윈도우의 작업 관리자와 같은 역할을 하는 그래픽 기반의 터미널 도구입니다.
```bash
$ htop
```
 * **체크 포인트:** 화면 상단의 CPU 코어별 막대그래프가 100%에 가깝게 채워져 있는지, Mem(메모리) 바가 가득 차 있는지 확인합니다.
 * **해결법:** 만약 특정 프로세스가 자원을 독점하고 있다면, 해당 행의 **PID(프로세스 ID)**를 확인한 후 kill -9 [PID번호] 명령어로 강제 종료할 수 있습니다. (htop 종료는 q 키)
### ② 디스크 용량 점검: df -h
저장장치(SD 카드 또는 SSD) 용량이 90% 이상 가득 차면 시스템 전체가 급격하게 느려집니다.
```bash
$ df -h
```
 * **체크 포인트:** 최상위 뿌리 경로인 / (루트)의 Use% (사용률)를 확인합니다. 여유 공간이 없다면 가상 메모리 공간이 부족해져 버벅임이 심해집니다.
### ③ CPU 온도(발열) 체크: vcgencmd (라즈베리파이 전용)
기기의 온도가 너무 높으면 하드웨어를 보호하기 위해 스스로 성능을 낮추는 '쓰로틀링(Throttling)'이 걸립니다.
```bash
$ vcgencmd measure_temp
```
 * **체크 포인트:** 대기 상태에서는 40~~50도, 작업 시 60~~70도가 정상입니다. 만약 80도에 육박한다면 쿨링팬이나 통풍 상태를 즉시 점검해야 합니다.
## 2단계: 범인이 '크로미움 브라우저'일 때 진단법
만약 자원 점검 결과 CPU나 메모리가 꽉 찬 원인이 크로미움 브라우저 때문이라면, 아래 두 가지를 확인해야 합니다.
### ① 크로미움 자체 작업 관리자 확인
크로미움 창 안에서 **Shift + Esc**를 누르면 브라우저 전용 작업 관리자가 뜹니다. 무한 루프가 돌거나 렉을 유발하는 특정 탭이나 확장 프로그램을 찾아 **[프로세스 종료]**를 누르면 시스템이 즉시 가벼워집니다.
### ② 하드웨어 가속(GPU) 상태 점검
크로미움 주소창에 **chrome://gpu**를 입력하고 엔터를 누릅니다.
 * Canvas, Compositing, Rasterization, WebGL 항목들이 **Hardware accelerated**(녹색)로 표시되어야 정상입니다.
 * 만약 Software only(적색)로 되어 있다면 CPU가 모든 그래픽 연산을 도맡아 하느라 느려진 것입니다. 브라우저 설정 ➔ 시스템 ➔ **[가능한 경우 그래픽 가속 사용]**을 켜고 브라우저를 재시작해 주세요.
## 3단계: 크로미움 속도 부스팅 해결 방법
하드웨어 설정에 문제가 없는데도 무겁고 느리다면 다음 3가지 조치를 취해줍니다.
 1. **인터넷 사용 기록 및 캐시 삭제:** **Ctrl + Shift + Delete**를 눌러 오랫동안 쌓인 웹 캐시 파일들을 한 번 시원하게 비워줍니다.
 2. **메모리 세이버(Memory Saver) 활성화:** 크로미움 설정 ➔ 성능 메뉴에서 '메모리 세이버'를 켜면, 현재 보지 않고 열어두기만 한 탭의 메모리를 자동으로 확보해 줍니다.
 3. **백그라운드 앱 실행 끄기:** 설정 ➔ 시스템 메뉴에서 'Chromium이 종료되어도 백그라운드 앱을 계속 실행' 옵션을 꺼서 불필요한 자원 낭비를 막습니다.
### 💡 만능 치트키: 크로미움 프로필 강제 초기화
설정이 이것저것 복잡하게 꼬였거나 어떤 방법으로도 해결되지 않을 때는, 터미널에서 프로필 폴더를 완전히 날려 처음 설치했던 깨끗한 상태로 리셋하는 것이 가장 빠릅니다.
```bash
$ rm -rf ~/.config/chromium
```
*(⚠️ 주의: 저장된 북마크나 패스워드가 모두 지워지므로 실행 전 중요한 데이터가 있는지 반드시 확인하세요.)*
## 📌 요약 가이드

| 명령어 / 단축키 | 기능 | 목적 |
| :--- | :--- | :--- |
| **htop** | 실시간 자원 모니터링 | CPU/메모리 범인 검거 및 강제 종료 |
| **df -h** | 디스크 여유 공간 확인 | 저장장치 포화 상태 점검 |
| **Shift + Esc** | 크로미움 작업 관리자 | 브라우저 내 무거운 탭 찾아 종료 |
| **chrome://gpu** | 그래픽 가속 상태 확인 | GPU 하드웨어 가속 정상 작동 검사 |

하드웨어 점검부터 소프트웨어 최적화까지, 터미널 명령어를 알면 느려진 컴퓨터도 당황하지 않고 완벽하게 컨트롤할 수 있습니다.
