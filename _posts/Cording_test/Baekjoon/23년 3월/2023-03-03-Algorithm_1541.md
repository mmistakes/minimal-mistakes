{: .notice--primary}

### 문제

세준이는 양수와 +, -, 그리고 괄호를 가지고 식을 만들었다. 그리고 나서 세준이는 괄호를 모두 지웠다.

그리고 나서 세준이는 괄호를 적절히 쳐서 이 식의 값을 최소로 만들려고 한다.

괄호를 적절히 쳐서 이 식의 값을 최소로 만드는 프로그램을 작성하시오.

### 입력

첫째 줄에 식이 주어진다. 식은 ‘0’~‘9’, ‘+’, 그리고 ‘-’만으로 이루어져 있고, 가장 처음과 마지막 문자는 숫자이다. 그리고 연속해서 두 개 이상의 연산자가 나타나지 않고, 5자리보다 많이 연속되는 숫자는 없다. 수는 0으로 시작할 수 있다. 입력으로 주어지는 식의 길이는 50보다 작거나 같다.

### 출력

첫째 줄에 정답을 출력한다.

### 예제 입력 1

```
55-50+40
```

### 예제 출력 1

```
-35
```

### 예제 입력 2

```
10+20+30+40
```

### 예제 출력 2

```
100
```

### 예제 입력 3

```
00009-00009
```

### 예제 출력 3

```
0
```

---


### 문제를 풀면서 막혔던 부분


"-"를 만날 때 큰 수를 빼야 된다. "-" 기호를 만날 때 다음 "-" 까지, 다음 "-"가 없다면 끝까지 모든 수를 더해서 한 번에 빼 주면 된다.

### 코드

##### ver(1)

```python
a = input().split('-')
c = 0
for i in a[0].split('+'):
    c += int(i)
for i in a[1:]:
    for j in i.split('+'):
        c-=int(j)
print(c)

```
