# Columns and Data Types

## Data Types

- Strings: Alphanumeric characters and text
  : - CHAR - Fixed number of characters
  : - VARCHAR - Variable number of characters up to a maximum length
  : - Other types for longer text

> CHAR & VARCHAR
>
> - CHAR: If the length of the string is less than set or fixed-length, then it is padded with extra memory space
> - VARCHAR: If the length of the string is less than the set or fixed-length, then it will store as it is without padded with extra memory spaces

> Storing Dates and Times
>
> March 9, 2023, 4:51PM
>
> - DATE &rarr; 2023-03-09
>   : EX) Birthday
> - DATETIME &rarr; 2023-03-09 16:51:00
>   : EX) Events, Reservations
> - TIMESTAMP &rarr; (saved when record is updated)
>   : EX) TIMESTAMP is great for recording when an order was placed!
