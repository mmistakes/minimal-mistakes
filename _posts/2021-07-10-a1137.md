# 문제
두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

# 입력
첫째 줄에 테스트 케이스의 개수 T가 주어진다.<br>

각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. A와 B는 콤마(,)로 구분되어 있다. (0 < A, B < 10)

# 출력
각 테스트 케이스마다 A+B를 출력한다.

# 예제 입력 1 
5<br>
1,1<br>
2,3<br>
3,4<br>
9,8<br>
5,2<br>
# 예제 출력 1 
2<br>
5<br>
7<br>
17<br>
7<br>

# 코드

```c
#include <stdio.h>

int main() {
	int a, b, c;
	scanf("%d", &a);
	for (int i = 0; i < a; i++)
	{
		scanf("%d,%d", &b, &c);
		printf("%d\n", b + c);
	}
}
```
