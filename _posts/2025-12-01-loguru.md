loguru 로 logger 객체를 가져오기(객체 안 만들어도 됨, 전역 객체가 준비됨) 


```python
from loguru import logger 
```

콘솔 출력 설정


```python
# 1. 기본 핸들러 제거 (기본적으로 콘솔 출력을 담당하는 핸들러)
logger.remove()

# 2. 콘솔 출력을 TRACE 레벨로 다시 추가 (level="TRACE" 지정)
import sys
logger.add(sys.stderr, level="TRACE") 
# *sys.stderr는 콘솔(터미널) 출력을 의미합니다.*
```




loguru 설정하기 logger.add()


```python
logger.add(
    "logs/test.log", # 로그 저장 위치
    rotation="00:00", # 로그 파일 로테이션 시간(1일 단위로 새 로그 파일 생성)
    retention="1 week", # 로그 파일 삭제 범위(1주일 이후 로그 삭제)
    level="TRACE", # 로그 출력 레벨 지정(현재 TRACE 이상은 전부 출력(콘솔에는 모든 레벨의 log 가 다 보이지만 log 파일에는 TRACE 만 저장))
)
```




로그 출력 테스트


```python
logger.trace("1. TRACE: 매우 상세한 추적 정보")
logger.debug("2. DEBUG: 개발 및 디버깅용 상세 정보")
logger.info("3. INFO: 일반적인 시스템 정보")
logger.success("4. SUCCESS: 중요한 작업 성공 알림")
logger.warning("5. WARNING: 잠재적인 문제 경고")
logger.error("6. ERROR: 기능 실패 오류")
logger.critical("7. CRITICAL: 시스템 중단 가능 심각 오류")
```
