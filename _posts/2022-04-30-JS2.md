### 문제  

> 👨🏻‍💻 백준 2480번 
  ``` 
1에서부터 6까지의 눈을 가진 3개의 주사위를 던져서 다음과 같은 규칙에 따라 상금을 받는 게임이 있다. <br>
1.같은 눈이 3개가 나오면 10,000원+(같은 눈)×1,000원의 상금을 받게 된다. <br>
2.같은 눈이 2개만 나오는 경우에는 1,000원+(같은 눈)×100원의 상금을 받게 된다. <br>
3.모두 다른 눈이 나오는 경우에는 (그 중 가장 큰 눈)×100원의 상금을 받게 된다. <br>
예를 들어, 3개의 눈 3, 3, 6이 주어지면 상금은 1,000+3×100으로 계산되어 1,300원을 받게 된다. <br>
또 3개의 눈이 2, 2, 2로 주어지면 10,000+2×1,000 으로 계산되어 12,000원을 받게 된다. <br>
3개의 눈이 6, 2, 5로 주어지면 그중 가장 큰 값이 6이므로 6×100으로 계산되어 600원을 상금으로 받게 된다.<br>
3개 주사위의 나온 눈이 주어질 때, 상금을 계산하는 프로그램을 작성 하시오.
  ```

> 입력 
>> 첫쨰 줄에 3개의 눈이 빈칸을 두고 각각 주어진다. 
> 출력 
>> 첫째 줄에 게임의 상금을 출력한다. 
  
### 풀이   

> 👨🏻‍💻 엉망진창 풀이 
``` Javascript
class ThrowDice {
    chance(first,second,third){
    if (first === 3 && second === 3 && third === 6){
        console,log(1000+3*100)
    }if (first === 2 && second === 2 && third === 2){
        console,log(10000+2*1000)
    }if (first === 6 && second === 2 && third === 5){
        console,log(6*100)
    }
}
}
const throwDice = new ThrowDice()
const first = prompt("첫번째 주사위를 입력하세요")
const second = prompt("두번째 주사위를 입력하세요")
const third = prompt("세번째 주사위를 입력하세요")

console.log(throwDice.chance()).

// undefined
```
> 👨🏻‍💻 엉망진창 풀이 2
``` Javascript
class ThrowDice {
    chance(first,second,third){
    return new Promise((resolve, reject)=>{
        setTimeout(()=>{
            if(
                (first === 3 && second === 3 && third === 6)||
                (first === 2 && second === 2 && third === 2)||
                (first === 6 && second === 2 && third === 5)
            ){
                resolve(first)
            } else {
                reject(new Error('not found'))
            }
        },2000)
    })
}
}

const throwDice = new ThrowDice()
const first = prompt("첫번째 주사위를 입력하세요")
const second = prompt("두번째 주사위를 입력하세요")
const third = prompt("세번째 주사위를 입력하세요")

throwDice
.chance(first,second,third)
.then(console.log(10000+2*1000))
.then(console.log(1000+3*100))
.then(console.log(6*100))

// 주사위 3개가 같은 경우라 12000만 나와야 하는데 12000, 1300, 600 한번에 나옴 
```
