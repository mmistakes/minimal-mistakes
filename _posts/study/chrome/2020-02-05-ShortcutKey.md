해당 포스팅은 지속적으로 업데이트 될 수 있습니다.
{: .notice--success}

`Ctrl` + `Tab` -> 크롬 탭 이동(우)
`Ctrl` +  `Shift` + `Tab` -> 크롬 탭 이동(좌)
`Ctrl` + `T` -> 새 탭
`Ctrl` + `E` -> 검색
`Ctrl` + `W` -> 현재 탭 끄기
`Ctrl` + `R` -> 새로 고침
`Ctrl` + `N` -> 새 창
`Ctrl` + `Shift` +  `N` -> 새 시크릿 창
