# 기초-삼항연산

1063번
```cpp
#include <stdio.h>

int main()
{
	int a,b;
	scanf("%d %d",&a,&b);
	printf("%d",a>b ? a:b);
	return 0;
}
```

1064번
```cpp
#include <stdio.h>

int main()
{
	int a,b,c;
	scanf("%d %d %d",&a,&b,&c);
	printf("%d",(a<b ? a:b)<c ? (a<b ? a:b):c);
	return 0;
}
```
