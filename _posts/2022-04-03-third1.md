맵함수를 이용하면 간단해지는데
map(function,iterable)

```python
def solution(n):
	nList = list(map(int, str(n)))
    newL = sorted(nList, reverse = True)
    strings = [str(i) for i in newL]
    a_string = "".join(strings)

    return int(a_string)
```

```python
def solution(n):
    ls = list(str(int(n)))
    ls.sort(reverse = True)
    return int("".join(ls))
```
