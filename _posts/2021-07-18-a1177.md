# 문제 설명
0~100 사이의 정수가 하나 입력될 경우 아래 규칙에 따라 적절한 메세지를 출력하는 프로그램을 작성하자.<br>


1. 100 : Perfect! Congratulations!<br>
2. 90 ~ 99: Very good!<br>
3. 80 ~ 89 : Good!<br>
4. 70 ~ 79 : Not bad!<br>
5. 0 ~ 69 : Study hard!<br>
# 입력 설명
0에서 100 사이의 정수가 하나 입력된다.
# 출력 설명
점수에 따른 정해진 메세지를 출력한다.
# 입력 예시 
85
# 출력 예시 
Good!
# 코드
```c
#include<stdio.h>

int main() {
    int a;
    scanf("%d", &a);
    if (a == 100) {
        printf("Perfect! Congratulations!");
    }
    else if (a <= 99 && a >= 90)
    {
        printf("Very good!");
    }
    else if (a <= 89 && a >= 80)
    {
        printf("Good!");
    }
    else if (a <= 79 && a >= 70)
    {
        printf("Not bad!");
    }
    else if (a <= 69 && a >= 0)
    {
        printf("Study hard!");
    }
}
```
