### 문제 링크
* http://icpc.me/14489

### 문제 출처
* 2017 선린 봄맞이 대회 C번

### 시간복잡도
* O(1)

### 풀이
두 통장의 잔고의 합은 a + b, 치킨 두 마리의 가격은 2c입니다.<br>
a + b가 2c보다 크거나 같으면 치킨을 살 수 있으므로 a + b - 2c를, 반대의 경우에는 a + b를 출력해줍니다.

### 전체 코드
```cpp
#include <stdio.h>

int main(){
	int a, b, c;
	scanf("%d %d %d", &a, &b, &c);
	if(a+b >= 2*c) printf("%d", a+b-2*c);
	else printf("%d", a+b);
}
```
