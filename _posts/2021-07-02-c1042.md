# 문제 설명
제 4항이 15이고 제 7항이 27인 등차수열이 있다고 할 때,  n번째 항이 얼마인지 출력하는 프로그램을 작성해 보자.

# 입력 설명
정수 하나가 입력된다.(입력값 범위 : 1 ~ 1000)

# 출력 설명
주어진 등차수열의 n번째 항의 값을 출력한다.

# 입력 예시 
7

# 출력 예시 
27


# 코드

```c
#include <stdio.h>

int main() {
    int a = 3, b = 4, c;

    scanf("%d", &c);
    int v = c - 1;
    printf("%d", a + (b* v));
}

```
