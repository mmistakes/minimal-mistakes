### 문제 링크
* https://www.acmicpc.net/problem/2164

### 사용 알고리즘
* 큐

### 시간복잡도
* O(n)

### 풀이
https://justicehui.github.io/2018/06/10/BOJ2161.html<br>
이 문제와 거의 동일한 문제이니 이 글을 먼저 읽고 와주시기 바랍니다.<br>

바로 해결방법을 적어보겠습니다.
0. 마지막에 나올 값을 저장할 변수 ans 선언
1. n 입력
2. 1부터 n까지의 수를 queue에 삽입
3. ans에 q.front() 저장
4. q.pop()
5. q.front()의 값을 push
6. q.pop()

3번에서 ans에 q.front()를 저장하는 이유는 가장 최근에 나온 카드를 계속 갱신해서 마지막으로 나온 카드를 최종적으로 출력하기 위해 q.front() 를 저장합니다.

### 전체 코드
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    int n;
    int ans;
    scanf("%d", &n);
    queue<int> q;
    for(int i=1; i<=n; i++) q.push(i);
    while(!q.empty()){
        ans = q.front();
        q.pop();
        q.push(q.front());
        q.pop();
    }
    printf("%d", ans);
}
```
